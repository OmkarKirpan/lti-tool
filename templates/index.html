How It Works

1

Platform Initiates Login

OpenEdX sends an OIDC login request to the tool's login endpoint

2

Authentication Flow

Tool validates the request and redirects to platform's authentication endpoint

3

Launch with JWT

Platform sends signed JWT with user/course data to the launch endpoint

4

Access Granted

Tool validates JWT and grants access based on user's role and context
